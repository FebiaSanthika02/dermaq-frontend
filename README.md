# Frontend — SkinAI Website

## Setup

```bash
cd frontend
npm install
```

## Development

```bash
npm run dev
# Buka http://localhost:5173
```

## Build Production

```bash
npm run build
# Output di folder dist/
```

## Environment Variable

Buat file `.env.local` jika backend tidak di localhost:
```
VITE_API_URL=http://localhost:8000
```
